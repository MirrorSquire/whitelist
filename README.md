## StakeWise whitelist manager

# Getting Started
## Setup
- Setup docker.
- Create an env file `cp .env.local .env`
- Build an image `docker build . -t whitelist`

## Run
- `docker run -p 8000:8000 whitelist`
- Go to http://0.0.0.0:8000/
